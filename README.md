# ToolBox

> - 个人开发的一款常用的小工具箱微信小程序，纯净无广告。

## 功能介绍

### 1.根据时间段，自动换肤

### 2.手持弹幕-- 可调整字体大小、颜色、背景、滚动速度

### 3.设备充电 CSS 特效
